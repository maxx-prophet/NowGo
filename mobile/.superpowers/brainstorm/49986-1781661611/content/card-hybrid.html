What each element does

TOP SECTION
  • Colored dot = category
  • Event name (bold, 2 lines max)
  • Emoji + venue + neighborhood
  • Hook in italics (hidden if null)
BOTTOM BAR
  • Price · Time · Contextual label
  • Availability badge (right)
  • Leave by time or ⚡ Leave NOW
  • Wraps if needed on small screens
