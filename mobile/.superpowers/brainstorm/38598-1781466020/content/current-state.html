Known Issues

  • Count label text gets cut off when result number is large (e.g. "124 events tonight · ne...")
  • No budget filter — can't filter by Free / $25 / $50 / $100
  • No sort control — no way to sort by Best Match / Soonest / Nearest
  • No Walk-ins only toggle
  • Transit mode chips feel disconnected from the rest of the filters
  • No time window — always shows "tonight" with no narrowing
  • 6 category options only — spec calls for 12

What the Spec Calls For

  • Budget chips: Free / <$25 / <$50 / <$100 / Any
  • Category pills: All + 11 categories
  • Walk-ins only toggle
  • Sort toggle: Best Match / Soonest / Nearest / Cheapest
  • Location (GPS auto-detect, already wired)
  • Time window field
